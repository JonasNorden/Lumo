Temporary Reference Demo: Object Selection + Delete + Undo/Redo

Standalone sandbox for editor-v2 only. This file intentionally does not import or reuse the current editor-v2 interaction/history code.

Click an empty cell to place an object in the active layer. Click an existing object to select it. Press Delete to remove the selected object. Press Ctrl/Cmd+Z to undo and Ctrl/Cmd+Shift+Z to redo.

    Educational notes. State lives in one plain object: active layer, selected object id, object collections by layer, the next stable id, and two global history stacks. Place/delete actions record the full object snapshot needed for inversion. Undo pops one action, applies its inverse immediately, and pushes the original action onto the redo stack. Redo reapplies that same action and pushes it back onto undo.
