Entity parameters

Sätt parametrar för just den här instansen. Tomt = default. Värden tolkas smart (tal, true/false, null, JSON).
FogVolume — vad varje parameter gör
Ändra värden och se preview direkt här (utan export/runtime).
Preview visar falloff i höjd + opacity
Area
x0/x1 = start/slut (px). y0 = baslinje/mark (px). falloff = taper-längd från båda ändar i höjd + opacity (px).
Look
density = täthet/opacity (lägre = mer genomskinlig). lift = flyttar dimman upp/ner. thickness = hur hög dimman blir. layers = mjukhet (fler = mjukare men dyrare). drift = sidrörelse (0 = stilla).
Smoothing
diffuse = spridning i sidled. relax = hur snabbt den lugnar sig. visc = dämpning (högre = mindre fladder).
Interaction
radius = påverkanens bredd. push = clear bakom. bulge = bukt fram. gate = min-hastighet innan det triggar. Reaktion sker bara när Lumo är inne i volymen.
Organic
Subtil “andning”. strength/scale/speed. Sätt strength:0 för helt statisk dimma.
Render
lumoBehindFog = om Lumo ska synas bakom dimman (true/false).
Snabbt: Lägre dimma → sänk look.thickness eller höj look.lift negativt. Tunnare dimma → sänk look.density. Mer falloff i höjd → höj area.falloff.
ParameterValue
JSON (full params)
Tips: hålla strukturen intakt (area/look/smoothing/interaction/organic/render). Tomt = default.
